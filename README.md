# ACNP

> Autonomous Campus Navigation Platform

ACNP is a modular, middleware-agnostic autonomous navigation platform for autonomous vehicles operating in private campuses and similar environments.

## Vision
Build a reusable software platform with clear architectural boundaries between perception, localization, planning and vehicle control.

## Repository Roadmap
- docs/ : Software Architecture Specification (SAS)
- src/ : Future implementation
- simulation/ : Digital twin and simulation assets
- interfaces/ : Interface definitions
- tests/ : Validation
